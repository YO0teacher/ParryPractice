Deadlock parry practice app. Randomly plays heavy melee sound effects that you can parry by pressing F.
Users with the same min & max trigger times will get parry events at the same time.
Optionally, add your own reaction images to the Image folder. Supported file types: .png, .jpg, .jpeg, .gif, .webp
The "Trigger practice" button triggers a parry event in the next 1-2 seconds. This only works while status is Disabled and does not increase your Total Parries or Streak score.
